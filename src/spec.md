# Specification

## Summary
**Goal:** Display UTC timezone offset alongside the local time in the Map Settings panel.

**Planned changes:**
- Append UTC offset in format "(UTC +X)" or "(UTC -X)" to the local time display in the WebsiteLayoutPanel component
- Update the specific local time element to show format like "February 24, 2026 at 9:11 AM (UTC +8)"

**User-visible outcome:** Users can see the UTC timezone offset next to the local time in the Map Settings panel, making it clear what timezone the displayed time represents.
